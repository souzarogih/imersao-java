Atividade Imersão Java 

Segue a descrição das atividades, se quiser pode usar o Scanner para 
facilitar a entrada de dados em qualquer uma das atividades.

1 - Faça um programa que mostre o dobro e o triplo de um número.
2 - Faça um programa que verifica qual é o maior números dentre dois.
3 - Faça um programa que faça os cálculos matemáticos com todos os operadores aritméticos visto em aula.
4 - Faça um sistema que calcule a média de um aluno e diga se ele foi aprovado.

5 Sistema de Desconto

Crie um programa que:
Declare uma variável double valorCompra
Se o valor for maior que 200 reais, aplique 10% de desconto
Mostre o valor final
Informe se o cliente recebeu desconto (boolean)
Dica: desconto de 10% = valorCompra * 0.10

6 Sistema de Idade para Dirigir

Crie um programa que:
Declare int idade
Verifique se a pessoa pode dirigir (idade >= 18)
Armazene o resultado em boolean podeDirigir
Mostre mensagem adequada no console

7 Cálculo de IMC (bem simples)

Crie um programa que:
Declare double peso
Declare double altura
Calcule o IMC: peso / (altura * altura)
Se IMC >= 25 → mostre “Acima do peso”
Caso contrário → “Peso normal”

8 Sistema de Par ou Ímpar

Crie um programa que:
Declare int numero
Verifique se é par ou ímpar usando %
Mostre o resultado
Dica: numero % 2 == 0

9 Sistema de Salário com Bônus

Crie um programa que:
Declare double salario
Declare int anosEmpresa
Se tiver mais de 5 anos → ganha bônus de 20%
Caso contrário → bônus de 10%
Mostre o salário final

10 Sistema de Estoque

Crie um programa que:
Declare int quantidadeEstoque
Se for menor que 10 → mostrar “Estoque baixo”
Caso contrário → “Estoque normal”

11 Sistema de menu de atendimento 

Crie um programa em Java que apresente o seguinte menu no console:
===== MENU DE ATENDIMENTO =====
1 - Informações sobre produtos
2 - Falar com atendente
